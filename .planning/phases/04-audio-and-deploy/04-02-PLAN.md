   **Brush-stroke SFX during kanji fade-in:**
   The current animation fades the entire SVG in over 0.8s (line 153-157). There are 24 stroke paths in KANJI_STROKES. To evoke the feeling of individual brush strokes being drawn, add staggered `playSfx("brush-stroke")` calls during the fade-in. In the main timeline, BEFORE the existing `tl.fromTo(kanjiSvg, ...)` fade-in, insert a series of `.call()` entries:

   ```typescript
   // Stagger brush-stroke SFX across the 0.8s fade-in
   // Play 4-5 brush strokes at intervals to suggest calligraphy without overwhelming
   const brushStrokeTimes = [0, 0.15, 0.3, 0.5, 0.65];
   brushStrokeTimes.forEach((time) => {
     tl.call(() => playSfx("brush-stroke"), [], time > 0 ? `+=${time - (brushStrokeTimes[brushStrokeTimes.indexOf(time) - 1] || 0)}` : "+=0");
   });
   ```

   ALTERNATIVELY (simpler and more robust): Add the brush-stroke calls as part of the fade-in by using `tl.call()` positioned at absolute times within the timeline. The cleanest approach is:

   ```typescript
   // Fade in kanji gradually
   tl.fromTo(
     kanjiSvg,
     { opacity: 0 },
     { opacity: 1, duration: 0.8, ease: "power1.inOut" }
   );

   // Overlay brush-stroke SFX during the fade-in period
   // Position these calls at the START of the fade (they overlap the fromTo above)
   const brushCount = 5;
   for (let i = 0; i < brushCount; i++) {
     tl.call(() => playSfx("brush-stroke"), [], (i * 0.8) / brushCount - 0.8);
     // Negative position = relative to current end, placing them DURING the fade-in
   }
   ```

   The exact positioning approach: use `tl.call(() => playSfx("brush-stroke"), [], "<+=0")` style positioning to overlap with the fade-in. Test different approaches and use whichever correctly fires during (not after) the 0.8s fade. The goal is 4-5 brush-stroke sounds spaced ~0.15s apart during the kanji reveal.

   **Ink-drop SFX after kanji fully visible:**
   After the fade-in completes and before the hold/splitTl, add:

   This represents the ink settling onto the surface after the brush strokes complete.
