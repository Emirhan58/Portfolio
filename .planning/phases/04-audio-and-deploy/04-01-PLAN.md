     - TRACKS array: `[["/audio/music/ambient-01.ogg", "/audio/music/ambient-01.mp3"], ["/audio/music/ambient-02.ogg", "/audio/music/ambient-02.mp3"]]`
     - SFX_NAMES: `["brush-stroke", "ink-drop", "katana-slash", "swoosh", "tink"]`

     - **Helper: `startMusicIfReady` callback:**
       Checks whether the kanji intro has completed before starting music. Logic:
       ```typescript
       const startMusicIfReady = useCallback(() => {
         if (!musicRef.current) return;
         const introSeen = sessionStorage.getItem("kanji-intro-seen");
         if (introSeen === "true") {
           // Intro already done — start music immediately with fade-in
           musicRef.current.play();
           musicRef.current.fade(0, volume, 1500);
         } else {
           // Intro still in progress — listen once for completion, then start music
           const onIntroDone = () => {
             if (musicRef.current) {
               musicRef.current.play();
               musicRef.current.fade(0, volume, 1500);
             }
           };
           window.addEventListener("kanji-intro-done", onIntroDone, { once: true });
         }
       }, [volume]);
       ```
       This ensures ambient music NEVER plays during the kanji intro per the locked decision: "Ambient music starts after intro completes (hero reveal), not during intro."

     - `useEffect` on mount: read `localStorage.getItem("audio-prefs")`, parse JSON `{ enabled, volume }`, set volume state. If `enabled` was true, register one-time `click`/`touchstart` listener on `document` that calls `initAudio()`, sets enabled true, then calls `startMusicIfReady()` (NOT `musicRef.current.play()` directly). Clean up listeners on unmount.
     - `initAudio` callback: if `initialized.current` return. Set `initialized.current = true`. Create SFX Howl instances for each name in SFX_NAMES with src `["/audio/sfx/{name}.ogg", "/audio/sfx/{name}.mp3"]`. Create music Howl with `html5: true`, `loop: true`, `volume: 0`. On music `end` event, advance `currentTrackIndex`, load next track from TRACKS array, play with fade-in.
     - `toggleAudio` callback: call `initAudio()`. Toggle `enabled`. If enabling: play `sfxRef.current["tink"]`, set `Howler.volume(volume)`, call `startMusicIfReady()` (NOT `musicRef.current.play()` directly — music must wait for intro completion). If disabling: call `musicRef.current.fade(volume, 0, 1000)`, setTimeout 1000ms then `musicRef.current.pause()`. Save to `localStorage.setItem("audio-prefs", JSON.stringify({ enabled: next, volume }))`.
     - `playSfx` callback: if `!enabled` or `!sfxRef.current[name]` return. Call `sfxRef.current[name].play()`. Add debounce: track last play time per SFX name, skip if < 200ms since last play.
     - `setVolume` callback: update volume state, call `Howler.volume(v)`, save to localStorage.
     - Cleanup `useEffect`: on unmount call `musicRef.current?.unload()`, iterate sfxRef and call `.unload()`. Also remove any lingering "kanji-intro-done" event listener.
     - Render: `<AudioCtx.Provider value={{ enabled, volume, toggleAudio, setVolume, playSfx }}>{children}</AudioCtx.Provider>`
   - Export `useAudio` hook: `useContext(AudioCtx)` with null check throwing `"useAudio must be inside AudioProvider"`.
